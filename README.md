# CALRADIA VOICES (EXTERNAL TTS INJECTOR)
### Фанатское дополнение для системы динамических диалогов в Bannerlord

---

## 1. ОБЩЕЕ ОПИСАНИЕ
Calradia Voices — это независимый программный инструмент (инжектор озвучки), разработанный для расширения атмосферы игры Mount & Blade II: Bannerlord. Программа работает в связке с модификацией AI Influence.

**Принцип работы:** Инструмент работает автономно от игры и мода. Он в реальном времени отслеживает системные логи и временные файлы данных (JSON), которые генерирует мод при общении с NPC. Как только файл обновляется, скрипт считывает текст реплики, обрабатывает его через нейросети и воспроизводит голос персонажа.

---

## 2. КЛЮЧЕВЫЕ ОСОБЕННОСТИ И ФУНКЦИИ

### Технология обработки текста (Фикс "NNN")
В логах мода часто встречаются технические символы (например, `\\n` для переноса строки). Обычные нейросети пытаются прочитать их буквально, произнося «эн-эн-эн». Данный инструмент содержит встроенный фильтр, который:
* Полностью вырезает программный мусор.
* Удаляет текст в звездочках (описания действий, например: *улыбается*).
* Корректно расставляет паузы на месте знаков препинания для естественного звучания.

### Система уникальности персонажей (Hash-Random)
Чтобы лорды не звучали одинаково, в скрипт встроена математическая модель рандомизации:
* **Индивидуальный отпечаток:** Скрипт берет уникальный ID персонажа из игры и превращает его в числовой код.
* **Герцовка (Pitch):** На основе этого кода каждому NPC назначается своя высота голоса (от баса до высокого тембра).
* **Темп речи (Rate):** Скорость произношения также подбирается индивидуально.

## 3. ВАРИАНТЫ ОЗВУЧКИ

### Вариант А: banner_voice.py (Edge-TTS) — Бесплатный
Использует облачные технологии Microsoft. Идеален для тех, кто хочет играть без ограничений.
* **Цена:** Полностью бесплатно.
* **Лимиты:** Отсутствуют.
* **Голоса:** Смешанный пул (мужские и женские голоса, включая мультиязычные движки с акцентами).

### Вариант Б: banner_voice_v2.py (ElevenLabs) — Премиальный
Подключает профессиональные нейросети через API.
* **Качество:** Максимально реалистичная дикция, человеческие вздохи и интонации.
* **Акценты:** Иностранные голоса (Andrew, Ava) читают русский текст с уникальным колоритом.
* **Требования:** Требуется персональный API-ключ и наличие символов на балансе ElevenLabs.

## 4. ИНСТРУКЦИЯ ПО УСТАНОВКЕ И НАСТРОЙКЕ

### Шаг 1: Подготовка окружения
1. Установите **Python 3.10** или выше с официального сайта python.org.
2. При установке обязательно отметьте пункт **"Add Python to PATH"**.
3. Откройте терминал (командную строку) и установите необходимые библиотеки командой:
   `pip install playsound==1.2.2 edge-tts requests`

### Шаг 2: Указание пути к данным (Директория)
Это самый важный этап. Вам нужно найти папку, куда мод AI Influence записывает файлы лордов. Обычно путь выглядит так:
`C:\Games\Bannerlord\Modules\AIInfluence\save_data\ВАШ_ID\`

1. Откройте выбранный скрипт через Блокнот.
2. Найдите строку `WATCH_DIR = r'...'`.
3. Вставьте ваш путь внутри кавычек после буквы `r`.
   *Пример: WATCH_DIR = r'C:\Users\Admin\Desktop\Bannerlord\save_data'*

### Шаг 3: Настройка ключа (только для Elite версии)
1. Зарегистрируйтесь на сайте ElevenLabs.
2. Скопируйте свой API Key в профиле.
3. Вставьте его в скрипте в строку `API_KEY = "ВАШ_КЛЮЧ"`.

## 5. ЗАПУСК И ИСПОЛЬЗОВАНИЕ

1. Сначала запустите скрипт (файл `.py`) двойным кликом.
2. Убедитесь, что в консоли появилось сообщение о готовности. (в версии banner_voice.py скрипт просто выводит версию скрипта)
3. Запустите игру Mount & Blade II: Bannerlord.
4. Подойдите к любому лорду и начните диалог.
5. Озвучка сработает автоматически в течение нескольких секунд после обработки текста на экране.

---

## 6. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

* **Скрипт пишет "Permission denied":** Это нормально, если игра одновременно со скриптом пытается перезаписать файл. Скрипт попробует еще раз через долю секунды.
* **Озвучка повторяется:** Скрипт имеет встроенный кэш, чтобы не повторять одну и ту же фразу. Если фраза повторилась, значит мод пересоздал файл с микро-изменением.
* **Ошибка Playsound:** Если скрипт вылетает при попытке проиграть звук, убедитесь, что вы установили именно версию `1.2.2`. Новые версии библиотеки работают в Windows некорректно.
